 Cross-Instance Alignment                                                                                                                                                                                                                  │
     │                                                                                                                                                                                                                                           │
     │ Context                                                                                                                                                                                                                                   │
     │                                                                                                                                                                                                                                           │
     │ Alignment discovery (interests, outcomes, interest maps) currently works within a single cooperative. A network — which is a cooperative whose members are other cooperatives — has no way to see or aggregate alignment data across its  │
     │ members. This is the remaining Stage 3 work.                                                                                                                                                                                              │
     │                                                                                                                                                                                                                                           │
     │ Goal: Let networks query alignment data from member cooperatives and generate cross-instance interest maps showing alignment/conflict at the organizational level.                                                                        │
     │                                                                                                                                                                                                                                           │
     │ Branch: feature/cross-instance-alignment from main.                                                                                                                                                                                       │
